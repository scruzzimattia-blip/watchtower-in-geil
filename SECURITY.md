# Sicherheitsrichtlinie

## Unterstuetzte Versionen

Aktuell werden nur Patches fuer die neueste Hauptversion bereitgestellt.

| Version | Unterstuetzt |
| ------- | ------------ |
| 0.6.x   | ✅ Ja         |
| < 0.6.0 | ❌ Nein       |

## Meldung einer Sicherheitsluecke

Bitte sende Sicherheitsluecken nicht ueber öffentliche Issues. Melde diese stattdessen direkt per E-Mail an [mattia@scruzzi.com].

Wir bemuehen uns, innerhalb von 48 Stunden auf Meldungen zu reagieren und eine Loesung zu erarbeiten.
